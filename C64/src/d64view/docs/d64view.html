:: d64view ::

:: welcome ::

Welcome to the d64view home page! d64view shows d64 disk image contents, such as file directory, block availability map and error codes. It handles (for now) the d64 disk image format.

:: download ::

d64view is Open Source; source code and a win32 binary is available here:
source: d64view-0.99.tar.gz (185 kb)
win32 binary: d64view-0.99-win32.exe (277 kb)

d64view is done using wxWindows and should be portable to Linux.
